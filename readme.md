The source code for [debgen.xyz](https://debgen.xyz)

# Setup

## Install source code

```
git clone https://github.com/rwarasaurus/debgen.git .
```

## Install packages

```
yarn
```

or

```
npm install
```

## Starting dev server

### Dev mode

```
yarn dev
```

## Building code

```
yarn build
```
